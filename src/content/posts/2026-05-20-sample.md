## 这是一个占位示例

> 真实的文章由 [tg_chat_tool](https://github.com/CornPrincess/tg_chat_tool) 每日自动生成并 push 到 `src/content/posts/` 目录。

### 频道结构

每个 Telegram 群组每天会生成一篇摘要，命名格式：

```
src/content/posts/<YYYY-MM-DD>-<chat-slug>.md
```

### Frontmatter schema

由 `src/content/config.ts` 强校验：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | string | ✓ | 文章标题 |
| `date` | YYYY-MM-DD | ✓ | 摘要日期 |
| `chat` | string | ✓ | 群 slug |
| `tags` | string[] | | 默认 `[]` |
| `summary` | string | | 列表页 excerpt |
| `source` | string | | 默认 `tg-digest` |

### 删除此示例

第一篇真实文章产出后，本文件可以删除。
